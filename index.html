Bienvenidos

Bienvenidos a DevStars 🚀, una agencia web ficticia creada como parte de nuestro Trabajo Práctico Grupal. Este proyecto refleja nuestro aprendizaje en desarrollo frontend, donde aplicamos buenas prácticas utilizando HTML, CSS y JavaScript. Nuestro objetivo fue diseñar una experiencia clara, moderna y responsive, permitiendo una navegación sencilla entre las distintas secciones y perfiles del equipo.

Proyecto

DevStars es una agencia web ficticia creada como parte de un trabajo práctico grupal. Nuestro objetivo es diseñar y desarrollar una página web moderna, responsive y estructurada aplicando buenas prácticas de desarrollo frontend.

Servicio

Desarrollo Web

Creamos sitios web a medida que combinan diseño moderno, rendimiento optimizado y una experiencia de usuario fluida. Desde páginas institucionales hasta tiendas online, cada proyecto está pensado para representar tu marca y convertir visitas en resultados reales.

Diseño UX/UI

Diseñamos experiencias digitales intuitivas que guían al usuario de forma natural. Analizamos comportamientos, estructuramos interfaces y creamos prototipos funcionales que equilibran estética y usabilidad para lograr productos claros, eficientes y atractivos.

Diseño Gráfico

Desarrollamos identidades visuales que comunican con fuerza y coherencia. Desde logotipos hasta piezas digitales y publicitarias, trabajamos cada detalle para que tu marca tenga presencia, personalidad y reconocimiento en cualquier entorno.

Mantenimiento Web

Nos encargamos del mantenimiento continuo de tu sitio para que funcione de forma segura, rápida y sin interrupciones. Realizamos actualizaciones, optimizaciones y soporte técnico para que vos puedas enfocarte en tu negocio sin preocupaciones.

Integrantes

Somos cuatro estudiantes de la Tecnicatura Superior en Desarrollo de Software del IFTS N.°29, Buenos Aires. La presente pagina es nuestro primer proyecto en equipo. Detrás de cada sección hay cuatro perspectivas distintas, horas de código y el desafío de construir desde cero una interfaz responsive y semántica usando solo HTML, CSS y JS, sin frameworks de por medio.
